# Android ISO Tool - Name Options

## Problem Solved
Emergency bootable USB creation from Android when computer is down

## Name Considerations
- Simple, memorable
- Conveys purpose
- BarrerSoftware branding

## Options
1. **FlashDroid** - Simple, clear (might conflict with existing?)
2. **BarrerFlash** - Branded, direct
3. **ISODroid** - ISO + Android obvious
4. **QuickBoot** - Emphasizes emergency use
5. **BootForge** - Implies creating bootable drives
6. **RescueDroid** - Highlights emergency recovery
7. **BarrerBoot** - Branded + purpose
8. **ISOForge** - Creating/forging ISOs to USB

## My Preference
**BarrerFlash** or **ISOForge** - both clear, professional, memorable
